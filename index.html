Slf4s

Simple Logging Facade for Scala

Download this project as a .zip file Download this project as a tar.gz file

Installation

Build.scala

      libraryDependencies ++= Seq(
        "org.slf4s" % "slf4s-api" % "1.7.5",
        "ch.qos.logback" % "logback-classic" % "1.1.0"
      )

Or build.sbt

libraryDependencies += "org.slf4s" % "slf4s-api" % "1.7.5"

libraryDependencies += "ch.qos.logback" % "logback-classic" % "1.1.0"

Oh and...

You don't have to use logback, any slf4j compatible framework will do.

Usage

Logging Mixin (trait)

class ExampleSpec extends WordSpec with Logging {
  "Logging should be easy" in {
    val importantValue = 10
    log.debug(s"importantValue: $importantValue")
    val importantThrowable = new Throwable
    log.debug(s"importantValue: $importantValue", importantThrowable)
  }
}
